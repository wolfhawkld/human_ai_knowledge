六、2026 关键趋势

  1. Matryoshka 嵌入已成标配 — 变长向量 256-3072 维灵活切换
  2. 多模态 Embedding 主流化 — 图文统一向量空间
  3. Encoder 回归 — gte-modernbert-base (149M) 用 1/50 参数量逼近 7B
  4. 开源追赶闭源 — jina-v5/Apache 2.0 MTEB v2 超越多数闭源
  5. 域适应微调 > 选最强通用模型 — 垂类域适应提升远超模型间切换
  6. 自部署成本 5-20x 低于 API — TEI + GPU 性价比极高
