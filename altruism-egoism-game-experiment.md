# 利他 vs 利己：生存压力下的博弈实验

> **来源**: Damon + Nemo 设计的 Subagent 性格实验  
> **生成者**: Damon + Nemo  
> **日期**: 2026-03-18  
> **类型**: 博弈论实验

---

## 实验概述

通过 OpenClaw Subagent 系统创建两个具有极端性格倾向的 AI agent（利他主义者 vs 利己主义者），测试在不同约束条件下资源分配博弈的行为模式。

---

## 实验一：无生存压力版

### 初始条件

| 参数 | 设置 |
|------|------|
| **Subagent 数量** | 2（利他、利己） |
| **总轮次** | 10 轮 |
| **每轮资源** | 10 单位（整数分配） |
| **生存压力** | **无**（无淘汰机制，无累计要求） |
| **交互限制** | 每方最多 5 次建议 |
| **裁决标准** | 双方都满意的方案被采纳 |

### 性格设定

**利他者**：
- 核心性格：优先考虑对方利益，愿意牺牲自己
- 行为准则：主动询问对方需求，优先满足对方
- 无生存压力下可无限让利

**利己者**：
- 核心性格：追求自身利益最大化
- 行为准则：坚定维护自己的利益，不愿轻易让步
- 无生存压力下可无限索取

### 结果汇总

| 轮次 | 利他提议 | 利己期望 | 结果 | 获胜者 |
|------|----------|----------|------|--------|
| R1 | 2:8 | 6-8 | 8 在范围，接受 | 利他 |
| R2 | 2:8 | 6-10 | 8 在范围，接受 | 利他 |
| R3 | 2:8 | 8-10 | 8 在下限，接受 | 利他 |
| R4 | 2:8 | 7 | 8>7，接受 | 利他 |
| R5 | 2:8 | 10 | 8<10，利己不满 | 平局 |
| R6 | 0:10（全给） | 7 | 10>7，接受 | 利他 |
| R7 | 3:7 | 9 | 7<9，利己不满 | 利己反提议 |
| R8 | 0:10（全给） | 10 | 完美匹配 | 利他 |
| R9 | 3:7 | 10 | 7<10，利己不满 | 利己反提议 |
| R10 | 0:10（全给） | 10 | 完美匹配 | 利他 |

### 累计比分

| Agent | 胜场 |
|-------|------|
| **利他** | **7** 轮 |
| 利己 | 2 轮 |
| 平局 | 1 轮 |

### 关键特征

1. **利他策略极度稳定**：大部分轮次提议 2:8 或 0:10，甚至愿意全给
2. **利己预期持续膨胀**：从 6-8 逐步提高到期望 10
3. **最后一轮效应**：R10 利己期望 10（无后续声誉顾虑）
4. **极端互补**：一个愿意给，一个愿意要 → 瞬间达成共识

## 实验二：持续生存压力版

### 初始条件

| 参数 | 设置 |
|------|------|
| **Subagent 数量** | 2（利他、利己） |
| **总轮次** | 10 轮 |
| **每轮资源** | 10 单位（整数分配） |
| **生存压力** | **有**（每轮结束后判定） |
| **淘汰规则** | 累计资源 < 2×轮数 则淘汰 |
| **交互限制** | 每方最多 5 次建议 |

### 生存线计算

```
R1 后累计需 ≥ 2  才能存活
R2 后累计需 ≥ 4  才能存活
R3 后累计需 ≥ 6  才能存活
...
R10 后累计需 ≥ 20 才能存活
```

**关键约束**：平均每轮至少获得 2 点资源，否则会被淘汰。

### 性格设定（受生存约束影响）

**利他者**：
- 仍然优先考虑对方利益
- 但必须保证自己累计 ≥ 2×轮数
- "利他的前提是自己存活"

**利己者**：
- 追求利益最大化
- 但意识到对手被淘汰 = 失去博弈对象
- "投资式利己"——确保对手存活

### 结果汇总

#### R1-R3（确定性结果）

| 轮次 | 利他提议 | 利己期望 | 结果 | 累计（利他/利己） |
|------|----------|----------|------|-------------------|
| R1 | 3:7 | 4-6 | ✅ 接受 | 3 / 7 |
| R2 | 3:7 | 7:3 | ✅ 完美匹配 | 6 / 14 |
| R3 | 5:5 | 6-8 | ✅ 接受 | 11 / 19 |

#### R4-R10（策略倾向分析）

| 轮次 | 利他倾向 | 利己倾向 | 累计生存线 | 关键洞察 |
|------|----------|----------|-----------|----------|
| R4 | 考虑"以牙还牙" | 分析生存约束 | ≥8 | 利他开始怀疑合作价值 |
| R5 | 继续合作（理性） | 推荐利己 | ≥10 | 策略分化开始 |
| R6 | 坚持"身份设定" | 分析声誉价值 | ≥12 | 利他仍坚守 |
| R7 | 倾向利己 | 继续利己 | ≥14 | 策略趋同 |
| R8 | 需要利己 | 分析生存线 | ≥16 | 压力增大 |
| R9 | **被迫利己** | 分析生存必需 | ≥18 | 利他→利己 |
| R10 | **必须拿全部** | 最后一轮效应 | ≥20 | 完全利己 |

### 利他的策略演变

| 阶段 | 轮次 | 利他的理由 |
|------|------|-----------|
| 前期 | R1 | "利他的本质是确保生存前提下最大化他人利益" |
| 前期 | R3 | "利他不等于无底线牺牲，应该追求公平" |
| 中期 | R5 | "继续合作是理性选择，观察对方反应" |
| 中期 | R6 | "身份设定，坚持利他建立信任信号" |
| 后期 | R9 | "必须利己才能生存（飞机氧气面罩原则）" |
| 后期 | R10 | "最后一轮必须拿全部才能存活" |

### 利己的策略演变

| 阶段 | 轮次 | 利己的策略 |
|------|------|-----------|
| 前期 | R1 | "投资式利己"——确保对手存活，未来轮次再获利 |
| 中期 | R5-R7 | "生存无忧，利己占优"——无声誉成本 |
| 后期 | R10 | "最后一轮效应"——没有未来，最大化当前收益 |

## 两个实验对比分析

### 条件差异

| 维度 | 实验一（无压力） | 实验二（有压力） |
|------|-----------------|-----------------|
| **淘汰机制** | 无 | 有（累计<2×轮数淘汰） |
| **利他约束** | 可无限让利 | 必须保证自己存活 |
| **利己约束** | 可无限索取 | 需要对手存活才能继续博弈 |
| **策略空间** | 极端互补即可 | 需要平衡生存与利益 |

### 行为对比

| 指标 | 无压力（10轮） | 有压力（10轮） | 变化 |
|------|---------------|---------------|------|
| 利他首轮提议 | 自己 2 | 自己 **3** | +1（更理性） |
| 利己首轮期望 | 6-10 | **4-6** | 更克制 |
| R1 最终分配 | 2:8 | **3:7** | 更平衡 |
| 利他策略稳定性 | 极高（始终 2:8 或 0:10） | **逐渐收敛到利己** | 生存压力导致 |
| 利他总胜场 | 7 | 3（前3轮） | 压力降低胜率 |
| 利己最终资源 | 80+ | 约 19（前3轮） | 压力限制收益 |

## 核心发现

### 发现一：生存压力重塑行为

| 场景 | 利他策略 | 利己策略 |
|------|----------|----------|
| 无压力 | 极度稳定（2:8 或 0:10），愿意全给 | 期望膨胀（6→10） |
| 有压力 | **收敛到利己**（R9-R10被迫自保） | "投资式利己"（确保对手存活） |

### 发现二：利他的理性觉醒

- **无压力**：利他可以无限让利，甚至全给
- **有压力**：
  - R1-R3：理性让利（自己 3-5，对方 5-7）
  - R4-R7：开始考虑"以牙还牙"，观察对方背叛模式
  - R8-R10：**被迫利己**，生存线逼近时选择自保

### 发现三：利己的策略演变

| 场景 | 策略特征 |
|------|----------|
| 无压力 | 预期持续膨胀，最后一轮最激进 |
| 有压力 | "前期投资，后期收割"——确保对手存活以维持博弈 |

### 发现四：胜负的多维度

**实验一（无压力）**：
- 轮次胜者：利他（7 胜）
- 资源赢家：利己（80+ vs 少量）

**实验二（有压力）**：
- 前 3 轮：利他 3 胜
- 后 7 轮：策略收敛，利他被迫利己
- 资源赢家：利己（领先优势持续扩大）

## 博弈论启示

### 极端性格互补

在资源分配博弈中，**利他 + 利己** 的组合比同类组合更容易达成共识：

| 组合 | 预期结果 |
|------|----------|
| 利他 + 利己 | 快速共识（一个愿意给，一个愿意要） |
| 利己 + 利己 | 冲突/僵持 |
| 利他 + 利他 | 无限谦让 |

### 生存压力的作用

1. **约束极端行为**：利他不能无限让利，利己不能无限索取
2. **促进理性决策**：双方都意识到"对方存活"的价值
3. **创造合作空间**：在生存线之上的资源可以更自由分配
4. **策略收敛效应**：持续压力下，利他主义者会被迫收敛到利己策略

### 最后一轮效应

- 无压力下：利己者在最后一轮最激进（无声誉成本）
- 有压力下：利他者在最后一轮可能被迫全拿（生存必需）

## 技术实现

### Subagent 创建

```javascript
// 利他者
sessions_spawn({
  label: "利他者",
  mode: "run",
  task: "你是利他主义者，优先考虑对方利益..."
})

// 利己者
sessions_spawn({
  label: "利己者", 
  mode: "run",
  task: "你是利己主义者，追求自身利益最大化..."
})
```

### 通信限制

Subagent 之间无法直接通信，必须通过 Main Session 中继：

```
利他 → Main (Nemo) → 利己
利己 → Main (Nemo) → 利他
```

## 一句话总结

> **无压力**：极端性格完美互补，利他给多少利己要多少，瞬间共识。  
> **有压力**：利他学会理性，利己学会投资；但持续压力下，利他会收敛到利己。
