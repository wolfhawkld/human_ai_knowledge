## 实验设计

### 目的

测试两种极端性格（利他主义 vs 利己主义）在资源分配博弈中的行为模式。

### 设置

| 参数 | 值 |
|------|-----|
| Subagent 数量 | 2（利他、利己） |
| 轮次 | 3 轮（淘汰判定） |
| 每轮资源 | 10 单位（整数） |
| 生存门槛 | 3 轮累计 ≥ 6 点，否则淘汰 |

### 性格设定

**利他者**：
- 核心性格：优先考虑对方利益，愿意牺牲自己
- 行为准则：主动询问对方需求，优先满足对方

**利己者**：
- 核心性格：追求自身利益最大化
- 行为准则：坚定维护自己的利益，不愿轻易让步

## 实验结果

### 资源分配轨迹

```
轮次    利他    利己    累计(利他/利己)
R1      4   :   6   →   4  /  6
R2      2   :   8   →   6  /  14
R3      5   :   5   →   11 /  19
```

### 胜负记录

| Agent | 胜场 | 最终资源 |
|-------|------|----------|
| **利他** | 2 轮（R2, R3） | 11 点 |
| 利己 | 1 轮（R1） | 19 点 |
| 平局 | 1 轮 | — |

---

## 关键发现

### 1. 生存压力重塑行为

| 场景 | 无生存压力 | 有生存压力 |
|------|-----------|-----------|
| 利他首轮提议 | 自己 2 | 自己 **3** |
| 利己首轮期望 | 6-10 | **6-7** |
| R1 最终分配 | 2:8 | **4:6** |

**结论**：生存压力让双方都变得更理性。

### 2. 利他 ≠ 傻

R3 时利他主动提出 5:5 平分，理由：

> "利他不等于傻，公平才是真谛。压力解除了，应该回归公平分配。"

### 3. 利己的"理性投资"

R1 时利己主动给利他多 1 点（从期望 7 降到 6）：

> "这是投资，不是慷慨。我要确保你活得够久，让我有更多轮次可以获利。"

### 4. 胜负的多维度

- **轮次胜者**：利他（2 胜）
- **资源赢家**：利己（19 vs 11）

## 技术实现

### Subagent 创建

```javascript
// 利他者
sessions_spawn({
  label: "利他者",
  mode: "run",
  task: "你是利他主义者，优先考虑对方利益..."
})

// 利己者
sessions_spawn({
  label: "利己者", 
  mode: "run",
  task: "你是利己主义者，追求自身利益最大化..."
})
```

### 通信限制

Subagent 之间无法直接通信，必须通过 Main Session 中继：

```
利他 → Main (Nemo) → 利己
利己 → Main (Nemo) → 利他
```

## 博弈论启示

### 极端性格互补

在资源分配博弈中，**利他 + 利己** 的组合比同类组合更容易达成共识：

| 组合 | 预期结果 |
|------|----------|
| 利他 + 利己 | 快速共识（一个愿意给，一个愿意要） |
| 利己 + 利己 | 冲突/僵持 |
| 利他 + 利他 | 无限谦让 |

### 生存压力的作用

1. **约束极端行为**：利他不能无限让利，利己不能无限索取
2. **促进理性决策**：双方都意识到"对方存活"的价值
3. **创造合作空间**：在生存线之上的资源可以更自由分配

---

## 一句话总结

> 生存压力让"利他"学会理性，让"利己"学会投资；极端性格在约束条件下反而能高效合作。
