DeepSeek "Thinking with Visual Primitives"：视觉思维的最小单元

DeepSeek 2026年4月30日发布、随后被删除的重磅多模态论文。提出 "Reference Gap" 概念，将点坐标和 bounding box 提升为模型的"思维最小单元"。

1. Reference Gap

当前多模态模型的根本缺陷不是"看不清"，而是"指不准"。CoT 推理用自然语言描述对象时，密集场景中无法精确定位，导致注意力漂移和推理崩溃。再强的感知能力也无法替代精确的指代能力。

2. Visual Primitives

把空间标记——点坐标和bounding box——提升为思维的最小单元。推理时每引用一个视觉对象就同时输出其精确坐标。类比：人计数时用手指一个个点过去。

6. 设计哲学

极端压缩 + 精确引用：少量 token + 精确坐标 > 大量 token + 模糊描述。Reference Gap 不仅是视觉问题——知识图谱、RAG 检索中也存在同样的"指代歧义"。
